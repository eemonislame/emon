Inputs and Outputs

scanf("format specifier"); Used for taking inputs.
printf("output"); Used for getting Outputs.
Now let's try an example.


Taking Input and getting Output:


#include<stdio.h>
int main()
{
    int a;
    scanf("%d", &a);
    printf("%d", a);
    return 0;
}

Explanation:

Here we used a as a variable, Variables are used to assign different kind of data. we used the variable a as integer with int , integer refers the data is a natural number which is contained by the variable.

Now the scanf("%d", &a);takes the value of the variable a from the user

And then printf("%d", a); displays the number/value of a as Output.

Basic Calculations

As we learned how to take inputs and how to get outputs, now we will try some basic calculations with the variables like summation, multiplication and dividing.

Sum:

To find the sum of two or more numbers, we have to use the plus(+)sign.And like this easy, we can find the sum of 7 and 5. Here's the code..


#include<stdio.h>
int main()
{
    int sum;
    sum = 7+5;
    printf("Result=%d",sum);
    return 0;
}
